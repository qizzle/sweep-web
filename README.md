# sweep-web

For the German [Jugend forscht](https://www.jugend-forscht-nrw.de/) competition.

## Tech Stack

This project uses [Astro](https://astro.build/) as the framework, paired with [TailwindCSS](https://tailwindcss.com/) for styling the site. \
[Bun](https://bun.com/) is used as the runtime instead of Node.js.
